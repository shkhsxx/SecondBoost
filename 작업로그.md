## [2026-07-02] PDF 추출

- 방법: 로컬 Google Chrome 헤드리스(`--headless=new --print-to-pdf`)로 `발표자료.html` → `발표자료.pdf` 변환
- 레이아웃 보존 원칙: 화면 표시용 CSS/디자인은 건드리지 않고, `@media print` 블록에만 규칙 추가
  - `.slide`를 `display:none` 토글 방식 대신 모두 표시 + `page-break-after:always`로 슬라이드당 1페이지 분리
  - `@page{ size: 13.333in 7.5in; margin:0; }`로 16:9(1280×720) 비율 그대로 유지 (기본 Letter 용지로 인한 비율 왜곡 방지)
- 검증: 생성된 PDF의 MediaBox가 960×540pt(16:9)로 9페이지 모두 일치함을 확인
